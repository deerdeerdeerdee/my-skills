# My Skills

Personal collection of Claude Code / Codex / Cursor skills.

## Available Skills

| Skill | Description |
|-------|-------------|
| [layered-documentation-governance](./layered-documentation-governance) | Maintain synchronized CLAUDE.md and AGENTS.md across project directories using a three-question test |

## Installation

### Via npx (recommended)

```bash
npx skills add deerdeerdeerdee/my-skills@<skill-name>
```

Example:
```bash
npx skills add deerdeerdeerdee/my-skills@layered-documentation-governance
```

### Manual Installation

```bash
git clone https://github.com/deerdeerdeerdee/my-skills.git
cp -r my-skills/<skill-name> ~/.claude/skills/
```

## Skill Compatibility

These skills are designed to work with:
- **Claude Code** (primary target)
- **Codex** (via AGENTS.md)
- **Cursor / Windsurf** (via AGENTS.md)
